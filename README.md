# 🌐 Aljazi Portfolio

A clean, modern personal portfolio website showcasing my journey as a Computer Science student, with a focus on Cybersecurity, Web Development, and Artificial Intelligence.

## ✨ Features
- Modern glassmorphism design with blurred, translucent cards
- Smooth scroll-reveal animations as sections come into view
- Fully responsive layout that adapts to any screen size
- Smooth scrolling navigation between sections
- Custom typography using Google Fonts (Outfit & Poppins)

## 📄 Sections
- **Hero** – Introduction and headline
- **About** – A short bio
- **Skills** – Interests and technical skills
- **Currently Learning** – What I'm studying right now
- **Summer Training** – AI & Robotics summer training program (2026)
- **Projects** – Project showcase (coming soon)
- **Contact** – Links to GitHub, LinkedIn, and Email

## 🛠️ Built With
- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- JavaScript (Intersection Observer API)

## 👩‍💻 About Me
Computer Science student at Taif University, expected to graduate in 2027, passionate about Cybersecurity, Web Development, and Artificial Intelligence.
